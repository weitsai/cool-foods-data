# 蔬時表 to Android String Array
因為參考來源的資料有些有空白，因此特別寫一個 php shell 去過濾空白，並且轉成 android xml 可以使用的 `string-array` 格式。

## 使用方式
1. Open `foods.php`
2. 把不需要的大類註解
3. 執行 foods.php
```
php foods.php
```
4. 把輸出的結果貼到 android 的 `<string-array>` 之間即可
